Getting Started

You want to contribute to Infinite Flight by editing airports? Awesome!

To edit airports for IF the following free software is required:

  • WorldEditor (or WED) by Laminar Research
  • SASPlanet or WED-O-Maker (depending on your OS)

To join the conversation with your fellow editors, we will invite you to Slack once you apply. You can use it in your web browser or as an app for PC/Mac and iOS/Android.

Furthermore, you will need a GitHub account.

Join us now by sending a mail to contact@airportediting.com or by filling out the form.

Please include the following information in the mail:

  • Your name
  • Your user name on the IF community if you have one
  • Your GitHub user name if you already have one
  • Your email address in case it's different from the one you are using for the mail

Please check out the beginner's guide here upon joining. All steps of the setup and editing itself are explained in the document.

We are looking forward to welcoming you!

Do's

Use accurate imagery for airports.
Add taxiways.
Add taxiway lines.
Add parking spots.
Modify runways.
Add airport boundaries.
Snap vertices.

Don'ts

Add runway markings with taxiway lines (this includes runway numbers, blastpad arrows, etc.).
Forget to include a boundary.
Make up an airport - it HAS to be a real airport.

Sending changes

In WorldEditor, go to File - Export as apt.dat. Save the file in the corresponding folder for the airport. Open up an email client (such as Gmail). Send an email to contact@airportediting.com.

forum

Slack

Join our Slack group to discuss with other editors! Let us know your email address via contact@airportediting.com. You'll also gain access to slash commands like /i-need-help, /wedomaker or /airport-info, which will provide you quick access to essential information.
